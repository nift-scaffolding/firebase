Firebase Scaffolding

   and

Nift project for scaffolding webapps with Google Firebase.

Getting Started

  • Fork and then clone the project using nift clone <clone-url>
  • Enter your Firebase Project ID in .firebaserc or run firebase use --add and select your Firebase project
  • Build the project using nift build
